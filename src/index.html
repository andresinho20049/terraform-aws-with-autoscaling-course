AWS Cloud Solutions Architect Study Project

This project is a hands-on study and demonstration of building scalable, modular, and production-ready AWS infrastructure using Terraform and Packer. The goal is to master best practices for the AWS Solutions Architect certification and real-world cloud deployments.

Infrastructure Overview

  • Multi-Environment Support: dev, staging, and prod with isolated state and configuration.
  • VPC: Custom networking with public/private subnets, route tables, and security groups.
  • Elastic File System (EFS): Shared storage for all application instances.
  • Application Load Balancer (ALB): Distributes traffic for high availability.
  • Auto Scaling Group (ASG): Ensures elasticity and resilience.
  • Bastion Host: Temporary, secure access for management operations.
  • Automated File Updates: Secure EFS updates via S3 and SSM, with automatic ASG refresh.
  • Infrastructure as Code: All resources managed and versioned with Terraform modules.
