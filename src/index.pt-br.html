Projeto de Estudo AWS Cloud Solutions Architect

Este projeto é um estudo prático e demonstração de como construir uma infraestrutura AWS escalável, modular e pronta para produção utilizando Terraform e Packer. O objetivo é dominar as melhores práticas para a certificação AWS Solutions Architect e para implantações reais em nuvem.

Visão Geral da Infraestrutura

  • Suporte a Múltiplos Ambientes: dev, staging e prod com estado e configuração isolados.
  • VPC: Rede customizada com subnets públicas/privadas, tabelas de rotas e security groups.
  • Elastic File System (EFS): Armazenamento compartilhado para todas as instâncias da aplicação.
  • Application Load Balancer (ALB): Distribui o tráfego para alta disponibilidade.
  • Auto Scaling Group (ASG): Garante elasticidade e resiliência.
  • Bastion Host: Acesso temporário e seguro para operações de gestão.
  • Atualização Automatizada de Arquivos: Atualizações seguras no EFS via S3 e SSM, com refresh automático do ASG.
  • Infraestrutura como Código: Todos os recursos gerenciados e versionados com módulos Terraform.
